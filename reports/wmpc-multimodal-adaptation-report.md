### 2.2 核心组件一：多模态状态表示 (Multimodal State Representations)

**问题**: 机器人需要从多种传感器（视觉、触觉、本体感觉等）获取信息。

**解决方案**:
- 统一的多模态编码器将不同传感器数据映射到共享潜在空间
- 支持视觉-触觉-本体感觉等多种模态的融合
- 与预训练基础模型对齐，利用大规模预训练知识

```
视觉观察 ──→ ┐
触觉反馈 ──→ ┼──→ 多模态编码器 ──→ 统一状态表示 z_t
本体感觉 ──→ ┘         ↑
语言指令 ──────────────┘
```

### 2.3 核心组件二：预训练可微世界模型

**关键洞察**: 与RL-based世界模型不同，WMPC使用预训练的可微世界模型，无需从头训练策略。

**世界模型功能**:
```
给定: 当前状态 z_t，动作 a_t
预测: 下一状态 z_{t+1} = f(z_t, a_t)
      奖励/成本 r_t = c(z_t, a_t)
```

**优势**:
- ✅ 利用预训练foundation模型的先验知识
- ✅ 可微分，支持梯度传播优化
- ✅ 可预测未来多步，支持MPC滚动优化

### 2.4 核心组件三：模型预测控制(MPC)优化

**滚动时域框架 (Receding Horizon)**:

```
目标函数:
min_{a_{t:t+H}} Σ_{i=0}^{H} c(z_{t+i}, a_{t+i}) + V(z_{t+H+1})

约束条件:
s.t.  z_{t+i+1} = f(z_{t+i}, a_{t+i})  (动力学约束)
      g(z_{t+i}, a_{t+i}) ≤ 0          (安全/物理约束)
      a_{t+i} ∈ 𝒜                      (动作空间约束)
```

**求解方法**:
- 利用世界模型的可微性进行梯度优化
- 支持约束感知优化（如CPO, Lagrangian方法）
- 实时求解，在线适应

```python
# WMPC算法伪代码

def WMPC(z_t, goal, pretrained_world_model):
    """
    输入:
        z_t: 当前多模态状态表示
        goal: 任务目标/成本函数
        pretrained_world_model: 预训练世界模型
    输出:
        a_t: 当前时刻最优动作
    """
    
    # Step 1: 弹性模型更新 (可选)
    if new_dynamics_detected():
        # 短期修正
        world_model = short_term_adaptation(pretrained_world_model, recent_trajectory)
        
        # 如果修正一致，更新长期记忆
        if consistency_check(world_model):
            pretrained_world_model = long_term_update(world_model)
    
    # Step 2: MPC滚动优化
    # 初始化动作序列
    a_seq = initialize_action_sequence()
    
    # 梯度优化
    for iter in range(num_opt_iterations):
        # 前向模拟
        z_sim, cost = rollout(world_model, z_t, a_seq, goal)
        
        # 反向传播优化
        grad = compute_gradient(cost, a_seq)
        a_seq = update_actions(a_seq, grad)
        
        # 投影到约束空间
        a_seq = project_to_constraints(a_seq)
    
    # 执行第一个动作
    a_t = a_seq[0]
    
    return a_t
```

## 7️⃣ 局限性与未来方向

### 7.1 潜在局限

1. **计算开销**: MPC的在线优化可能计算量大，需要高效求解器
2. **模型误差累积**: 长时间预测可能累积误差，需要不确定性建模

### 7.2 未来方向

1. **不确定性量化**: 引入概率世界模型，支持风险感知规划
2. **分层规划**: 结合高层符号规划和低层MPC控制
3. **多智能体扩展**: 扩展到多机器人协作场景
4. **硬件部署**: 边缘计算优化，支持实时嵌入式部署
