# π_RL: 基于流的视觉-语言-动作模型在线强化学习微调

**论文信息**: arXiv:2510.25889  
**标题**: Online RL Fine-tuning for Flow-based Vision-Language-Action Models  
**作者**: Kang Chen, Zhihao Liu, Tonghe Zhang, Zhen Guo, Si Xu, Hao Lin, Hongzhi Zang, Xiang Li, Bingwen Wei, Jiakai Zhou, Quanlu Zhang, Zhaofei Yu, Guoliang Fan, Tiejun Huang, Yu Wang, Chao Yu  
**机构**: 北京大学、清华大学等研究机构

---

## 📋 摘要

Vision-Language-Action (VLA) 模型使机器人能够理解和执行来自多模态输入的复杂任务。虽然近期研究探索使用强化学习(RL)来自动化监督微调(SFT)中繁琐的数据收集过程，但将RL应用于大规模基于流的VLA（如π₀、π₀.₅）仍然具有挑战性，原因在于流匹配产生的动作对数似然难以计算。

本研究通过 **π_RL** 解决这一挑战，提出两种技术方案：

1. **Flow-Noise**: 将去噪过程建模为离散时间MDP，使用可学习的噪声网络进行精确的对数似然计算
2. **Flow-SDE**: 将去噪过程与智能体-环境交互整合，构建双层MDP，采用ODE到SDE的转换实现高效的RL探索

实验表明，RL在分布内(ID)和分布外(OOD)设置中均带来显著的性能提升。

---

## 🎯 研究背景

### 什么是VLA模型？

视觉-语言-动作(VLA)模型是机器人领域的重要突破：
- **多模态理解**: 整合视觉图像、语言指令和机器人本体感知
- **动作生成**: 将抽象指令转化为可执行的物理动作
- **人机交互**: 实现直观灵活的交互方式

**代表模型**: π₀、π₀.₅、OpenVLA、OpenVLA-OFT、Octo、GR00T等

### 当前训练范式的局限

现有VLA模型通常采用 **预训练 → 监督微调(SFT)** 的流程：

1. **数据收集成本高昂**: 需要大规模、高质量的人类专家示范数据
2. **性能受限**: SFT模型容易过拟合到专家数据，性能受限于示范质量
3. **缺乏探索**: 无法通过环境交互持续改进

### Flow-based VLA的特殊挑战

π₀ 和 π₀.₅ 等基于**流匹配(Flow Matching)**的VLA模型：
- ✅ **优势**: 通过迭代精化生成动作，支持高频动作生成和执行精细任务
- ❌ **挑战**: 难以计算动作的对数似然(log-likelihood)，这是策略梯度方法的必需项

**核心问题**: 如何将强化学习应用于基于流的VLA模型？

## 🔬 核心方法

### 方案一：Flow-Noise

**核心思想**: 将去噪过程建模为离散时间马尔可夫决策过程(MDP)

#### 技术细节

1. **随机性注入**
   - 引入**可学习的噪声网络**，动态控制注入噪声的幅度
   - 去噪过程每一步建模为高斯分布：
     ```
     p(A^{τ+δ}|A^τ) ~ N(μ_τ, Σ_τ)
     ```
   - **均值**: 由原始ODE的前向欧拉更新确定
     ```
     μ_τ = A^τ + v^τ · δ
     ```
   - **方差**: 由噪声网络学习得到
     ```
     Σ_τ = diag(σ_θ'²)
     ```

2. **对数似然估计**
   - 将推理过程离散化为K个均匀步骤
   - 时间区间: δ = 1/K
   - 整个去噪序列的精确对数概率:
     ```
     log π(A|o) = log(π(A⁰|o) · ∏_{k=0}^{K-1} π(A^{τ_{k+1}}|A^{τ_k}, o))
     ```

### 方案二：Flow-SDE

**核心思想**: 将确定性ODE转换为随机微分方程(SDE)，构建双层MDP

#### 技术细节

1. **ODE转SDE**
   
   确定性ODE（Rectified Flow）:
   ```
   dA^τ = v^τ dτ
   ```
   
   转换为等价的SDE（保持边缘分布）:
   ```
   dA^τ = [v^τ + σ_τ²/(2τ)(A^τ + (1-τ)v^τ)]dτ + σ_τ dw_τ
            └─────────────────┬─────────────────┘   └──────┘
                  Drift Term          Diffusion Term
   ```
   
   其中噪声调度: σ_τ = a√(τ/(1-τ))

2. **双层MDP构建**
   
   | 层级 | 定义 | 组成 |
   |------|------|------|
   | **外层** | 环境交互 | 标准MDP，状态s_t，动作a_t |
   | **内层** | 去噪过程 | 状态s̄_t^τ = (o_t, A_t^τ)，动作ā_t^τ |

3. **混合ODE-SDE采样**
   - 随机选择某个去噪时间步进行SDE探索
   - 其余步骤使用确定性ODE更新
   - 有效缩短MDP视界，提升训练效率

### 策略优化

使用**近端策略优化(PPO)**算法进行训练：

#### Chunk-level公式
- π系列模型输出H个未来动作序列: A_t = [a_{t,0}, ..., a_{t,H-1}]
- 将整段动作视为单个大步
- 奖励: R_t = ∑_{j=0}^{H-1} r_{t,j}

#### 优势估计
使用广义优势估计(GAE):
```
Â_t = ∑_{k=0}^{T-t} (γλ)^k · T_{t+k}
```
其中TD误差: T_t = R_t + γV(s_{t+1}) - V(s_t)

#### PPO目标函数
```
J(π_θ) = E_t[min(ρ_t(θ)Â_t, clip(ρ_t(θ), 1-ε, 1+ε)Â_t)]
```

### Critic设计

采用共享的actor-critic架构：

- **π₀.₅**: Critic直接连接VLM输出，估计V_vlm(o_t)
- **π₀**: 由于动作专家需要噪声动作和状态作为输入，通过平均整个去噪轨迹的值来近似:
  ```
  V_expert(o_t) ≈ E_{τ~U[0,1]}[V_expert(o_t, A_t^τ)]
  ```

## 🧪 实验结果

### 测试基准

| 基准 | 任务特点 |
|------|----------|
| **LIBERO** | 长时序机器人操作任务 |
| **ManiSkill** | 4,352种拾取放置任务组合 |
| **MetaWorld** | 50种不同的操作原语 |
| **CALVIN** | 长时序序列任务 |

### 主要结果

#### 分布内(ID)性能

| 模型 | 方法 | LIBERO | ManiSkill | MetaWorld | CALVIN | 平均 | 提升 |
|------|------|--------|-----------|-----------|--------|------|------|
| **π₀** | SFT | 57.6 | 38.4 | 50.8 | 57.5 | 51.1 | — |
| | Flow-SDE | **96.1** | **78.8** | **78.1** | **61.7** | 78.7 | +27.6% |
| | Flow-Noise | **97.6** | **77.8** | **85.8** | **59.9** | 80.3 | +29.2% |
| **π₀.₅** | SFT | 77.1 | 40.1 | 43.8 | 61.3 | 55.6 | — |
| | Flow-SDE | **97.9** | **90.9** | **70.7** | **87.0** | 86.6 | **+31.0%** |
| | Flow-Noise | **98.3** | **89.7** | **66.1** | **84.5** | 84.7 | +29.1% |

**关键发现**:
- π_RL在所有基准上显著超越SFT基线
- π₀.₅ + Flow-SDE平均提升达 **+31.0%**
- 少样本SFT + RL可达到甚至超越完整数据集SFT的性能

#### 分布外(OOD)泛化

- ✅ **ManiSkill & CALVIN**: ID性能提升有效迁移到OOD场景
- ⚠️ **MetaWorld**: 在新任务上提升有限，说明RL主要增强动作级精细化而非跨任务泛化

### 消融研究

#### 1. Critic设计
- **VLM后接Critic**比Action Expert后接表现更好
- 4层MLP比1层MLP有更准确的价值估计

#### 2. MDP公式对比
| 方法 | 收敛速度 | 最终性能 | 计算效率 |
|------|----------|----------|----------|
| 单层MDP | 最快 | 相当 | 无优势（需重新计算完整轨迹） |
| 标准双层MDP | 中等 | 相当 | 基准 |
| 混合双层MDP | 中等 | 相当 | **2×加速** |

#### 3. 噪声注入策略
- 固定噪声和可学习噪声均有效
- 噪声水平权衡：
  - **低噪声(a=0.2)**: 训练不稳定，clip分数高
  - **高噪声(a=0.8)**: 探索能力强，但可能扭曲流轨迹

#### 4. 超参数敏感性

| 参数 | 关键发现 |
|------|----------|
| **去噪步数K** | K=1时SDE转换误差大；K≥2后性能稳定 |
| **动作块大小** | 更大的块减少交互频率，限制RL优化空间 |
| **噪声水平a** | 0.5左右取得最佳平衡 |

## 📝 总结

π_RL为Flow-based VLA模型的强化学习微调奠定了重要基础。通过Flow-Noise和Flow-SDE两种创新方案，成功解决了流匹配中的策略优化难题。实验结果表明，该方法在多个机器人操作基准上实现了显著的性能提升，为VLA模型的进一步发展和实际应用提供了有力支撑。
