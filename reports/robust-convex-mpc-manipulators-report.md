## 📌 核心贡献一句话总结

**本文提出了一种基于鲁棒Tube MPC和走廊规划的凸优化框架，通过反馈线性化获得线性预测模型并推导状态/输入依赖的误差上界，首次为不确定非线性机械臂构建了具有碰撞避免保证的凸MPC解决方案，在更高模型不确定性下实现更快更安全的运动。**

## 1️⃣ 研究背景与动机

### 1.1 工业机器人控制的挑战

工业机械臂通常在** cluttered environments（杂乱环境）**中操作，面临以下核心问题：

| 挑战 | 具体问题 | 现有方法局限 |
|------|----------|-------------|
| **模型不确定性** | 参数不确定导致动力学模型失配 | 传统方法被迫降速以减小扰动影响 |
| **碰撞避免** | 需要实时保证安全 | 非凸约束计算昂贵，难以实时求解 |
| **运动规划** | 路径规划+时间缩放+跟踪控制 | 解耦方法缺乏鲁棒性保证 |
| **实时性** | 需要1kHz或更高的控制频率 | 非线性MPC计算成本太高 |

### 1.2 现有方法的局限

**传统流水线**:
```
路径规划（采样法）→ 时间缩放优化 → 反馈线性化跟踪
```

**问题**:
1. 路径规划仅保证路径上无碰撞，偏离路径后无保证
2. 模型误差通过耦合动力学传播，导致状态/输入依赖的失配
3. 为补偿不确定性，只能**降低速度**
4. 缺乏有效的鲁棒性保证方法

### 1.3 核心洞察

> **模型误差具有状态/输入依赖性**：误差大小与速度、加速度密切相关。利用这一知识可以设计更少保守、更高效的控制器。

### 2.2 核心创新一：反馈线性化与状态依赖误差界

#### 动力学模型

机器人动力学方程：
```
u = M(q)q̈ + C(q,q̇)q̇ + g(q)
```

其中模型分为**名义（已知）**和**不确定**两部分：
```
M(q) = M₀(q) + M_θ(q)  ← 质量矩阵
C(q,q̇) = C₀(q,q̇) + C_θ(q,q̇)  ← 耦合矩阵（科氏力+阻尼）
g(q) = g₀(q) + g_θ(q)  ← 重力向量
```

#### 反馈线性化

使用反馈控制律：
```
u = π_FL(q,q̇,a) = M₀(q)a + C₀(q,q̇)q̇ + g₀(q)
```

得到闭环动力学：
```
q̈ = a + Δ_θ(q,q̇,a)
```

其中**模型误差项**：
```
Δ_θ(q,q̇,a) = M̃_θ(q)a + C̃_θ(q,q̇)q̇ + g̃_θ(q)
```

#### 关键创新：状态/输入依赖的误差上界

**命题1**: 对于所有状态和控制输入，模型误差满足：

```
‖BΔ_θ(x,a) + Δ_disc(x,a)‖_P ≤ β(x,a) := a‖a‖ + b‖q̇(x)‖ + c
```

**核心洞察**:
- **a项**: 与期望加速度成正比（惯性不确定性）
- **b项**: 与速度成正比（科氏力/阻尼不确定性）
- **c项**: 常数项（重力+离散化误差）

这允许控制器根据当前运动状态**自适应调整鲁棒性**。

### 2.3 核心创新二：Tube MPC与椭圆管动态

#### 辅助控制器

```
a = ā + K(x - x̄)
```

其中增益矩阵K和Lyapunov矩阵P离线求解，满足收缩条件：
```
‖(A+BK)x‖_P ≤ ρ‖x‖_P,  ρ ∈ (0,1)
```

#### 椭圆管定义

状态空间中的椭圆管：
```
ℰ(δ) = {x ∈ ℝ^{n_x} | ‖x‖_P ≤ δ}
```

投影到配置空间、速度空间、控制空间：
```
ℰ_q(δ) = {q(x) | x ∈ ℰ(δ)}       ← 配置空间管
ℰ_v(δ) = {q̇(x) | x ∈ ℰ(δ)}       ← 速度空间管
ℰ_a(δ) = {a = Kx | x ∈ ℰ(δ)}     ← 控制空间管
```

#### 管动态（命题2）

管大小演化的递推关系：
```
δ₊ = (ρ + L_β)δ + β(x̄, ā)

其中 L_β = a‖KP^{-1/2}‖ + b‖VP^{-1/2}‖
```

**稳态管大小**：
```
δ_f := c / (1 - ρ̃)    其中 ρ̃ = ρ + L_β
```

### 2.4 核心创新三：凸鲁棒MPC问题

#### 优化问题

```
min   Σ_{i=0}^{H-1} [‖x̄_i - x̄_H‖_Q² + ‖ā_i‖_R²] + ‖x̄_H - x_g‖_{Q_e}²

s.t.  ‖x̄_0 - x(k)‖_P ≤ δ_0                    ← 初始状态管约束
      x̄_{i+1} = Ax̄_i + Bā_i                  ← 名义动力学
      q̇(x̄_H) = 0, δ_H ≥ δ_f, x̄_H ∈ 𝒳⊖ℰ(δ_H+ε)  ← 终端约束
      δ_{i+1} ≥ ρ̃δ_i + β(x̄_i, ā_i)            ← 管动态
      x̄_i ∈ 𝒳⊖ℰ(δ_i)                          ←  tightened状态约束
      ā_i ∈ 𝒜⊖ℰ_a(δ_i)                        ←  tightened控制约束
```

**关键特性**:
- ✅ **凸优化**: 二阶锥规划(SOCP)，可高效求解
- ✅ **鲁棒性**: 通过管约束保证不确定系统的可行性
- ✅ **自适应**: 根据速度/加速度调整管大小
- ✅ **跟踪**: MPC-for-tracking框架，可到达远方目标

#### 约束收紧（Pontryagin差集）

原始约束被收紧以容纳管的不确定性：
```
𝒳⊖ℰ(δ) = {x | x ⊕ ℰ(δ) ⊆ 𝒳}
```

这确保真实状态始终在可行域内。

### 2.5 核心创新四：基于SCDF的走廊规划

#### SCDF（符号配置距离函数）

```
         ⎧ -min_{q_c ∈ ∂𝒞_o} ‖q - q_c‖   if q ∈ 𝒞_o (障碍物内)
r(q) =   ⎨
         ⎩ +min_{q_c ∈ ∂𝒞_o} ‖q - q_c‖   otherwise (自由空间)
```

#### 碰撞自由球

给定碰撞自由点c，定义碰撞自由区域：
```
ℬ(c) = {q ∈ ℝ^{n_c} | ‖q - c‖ ≤ r(c)} ⊆ 𝒞_f
```

#### 走廊规划算法

```
输入: 起始配置q_s, 目标配置q_g, 障碍物𝒪
输出: 控制序列

1. 离线学习SCDF近似
2. 使用采样法规划无碰撞路径γ(s): [0,1] → 𝒞_f
3. 离散化路径，计算碰撞自由球序列B = {(c_i, r_i)}
4. 迭代:
   a. 分配球给预测轨迹（根据当前位置选择最近的球）
   b. 选择虚拟目标x̃_g（沿路径最远可达点）
   c. 求解凸MPC（带碰撞避免约束）
   d. 并行执行辅助控制器n_a步
   e. 更新状态，重新规划
```

## 3️⃣ 理论保证

### 定理1（闭环性质）

对于不确定非线性系统，若MPC在k=0时可行，则：

1. **递归可行性**: 对所有k∈ℕ，问题保持可行
2. **约束满足**: x(k)∈𝒳, a(k)∈𝒜（状态和输入约束始终满足）
3. **收敛性**: limsup_{k→∞} {x(k) - x_g} ⊆ ℰ(δ_f)（收敛到目标附近）

### 相比现有方法的优势

| 特性 | 标准刚性Tube MPC | 本文方法 |
|------|-----------------|----------|
| **管形状** | 多面体不变集 | 缩放椭圆（更可扩展） |
| **误差界** | 常数（最坏情况） | 状态/输入依赖（更少保守） |
| **优化问题** | 非凸或计算昂贵 | 凸SOCP，实时求解 |
| **吸引域** | 较小 | 较大（任意稳态都是可行初始条件） |

## 7️⃣ 局限性与未来方向

### 7.1 局限性

1. **反馈线性化依赖**: 需要可逆的质量矩阵，某些欠驱动系统不适用
2. **SCDF近似**: 配置空间距离函数的学习/计算可能复杂
3. **参数不确定性假设**: 假设不确定参数有界，未处理非参数不确定性
4. **计算复杂度**: 虽然凸但维度高时求解时间可能增加

### 7.2 未来方向

1. **深度学习集成**: 用神经网络学习更复杂的误差模型
2. **多机器人协作**: 扩展到多机械臂协调控制
3. **柔性/软体机器人**: 适应更复杂的动力学
4. **在线参数估计**: 结合自适应控制，实时估计不确定参数
