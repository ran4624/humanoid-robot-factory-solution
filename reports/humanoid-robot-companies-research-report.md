# 全球人形机器人公司深度调研分析报告

> **报告日期**: 2026年3月20日  
> **分析师**: AI助手  
> **报告版本**: V1.0

---

## 执行摘要

人形机器人产业在2024-2025年迎来爆发式增长，全球融资规模超过40亿美元。本报告深度调研了**18家国内外主要人形机器人公司**，涵盖Figure AI、Tesla Optimus、宇树科技、优必选、星海图等行业头部企业，从人员规模、技术路线、融资估值、商业化进展等多维度进行全面分析。

**核心发现**:
- 全球人形机器人融资总额2024年超70亿元人民币，2025年预计超100亿元
- Figure AI以395亿美元估值领跑全球，宇树科技120亿元估值领跑国内
- 2025年被定义为"量产元年"，头部企业年交付量突破千台级别
- 技术路线分化：端到端VLA vs 分层模块化，电机驱动 vs 液压驱动

## 一、国外人形机器人公司

### 1.1 Figure AI（美国）

| 维度 | 详情 |
|------|------|
| **成立时间** | 2022年5月 |
| **总部** | 美国旧金山 |
| **创始人** | Brett Adcock（连续成功创业者） |
| **人员规模** | 约200-300人 |
| **组织形式** | 创业公司 |

**发展历程**:
- 2022年5月：公司成立
- 2023年3月：发布Figure 01原型机
- 2024年2月：完成6.75亿美元B轮融资，估值26亿美元
- 2024年8月：发布Figure 02，与宝马签订商业订单
- 2025年2月：终止与OpenAI合作，发布自研VLA模型Helix
- 2025年9月：完成10亿美元融资，估值飙升至**395亿美元**（约2800亿人民币）

**产品与技术**:
- **Figure 02**：第四代人形机器人，16自由度灵巧手，可抓取25kg物体，续航5小时
- **技术路线**：端到端VLA（Vision-Language-Action）模型Helix，自研大脑+小脑
- **应用场景**：汽车制造（宝马工厂）、仓储物流

**融资情况**:
| 轮次 | 时间 | 金额 | 估值 | 主要投资方 |

| A轮 | 2023 | - | - | - |

| C轮 | 2025.9 | 10亿美元 | 395亿美元 | Align Ventures、Parkway |

**累计融资**：超7.5亿美元（约52.6亿人民币）

#### 核心竞争力分析

**1. 顶配创始团队**
- **Brett Adcock**：连续成功创业者，此前创立Vettery（人才市场，被Adecco以1亿美元收购）和Archer Aviation（eVTOL，上市估值数十亿）
- **核心团队**：来自Google、Boston Dynamics、Tesla、DeepMind等顶级公司
- **人才密度**：200-300人团队，人均估值超1亿美元（行业最高）

**2. 技术路线：端到端VLA**
- **Helix模型**：2025年2月发布的自研VLA（Vision-Language-Action）模型
- **架构特点**：
  - 大脑（高层规划）：理解自然语言指令，生成长程任务计划
  - 小脑（低层控制）：200Hz实时控制，实现灵巧操作
  - 端到端训练：直接从像素+语言→动作，无需中间表示
- **性能表现**：
  - 能执行"拿起红色苹果放入篮子"等复杂指令
  - 双手协调操作，可抓取柔软、不规则物体
  - 在宝马工厂完成真实装配任务

**3. 商业模式：两条腿走路**
| 模式 | 内容 | 进展 |
|------|------|------|
| **商业试点** | 与宝马、亚马逊等合作，验证场景 | 宝马工厂部署中 |
| **技术授权** | 将Helix模型授权给其他机器人公司 | 探索中 |

**4. 争议与挑战**
- **与OpenAI分手**：2025年2月终止合作，从GPT-4切换至自研Helix
  - 原因：OpenAI模型响应速度慢（API调用延迟），无法满足实时控制需求
  - 影响：短期技术阵痛，长期自主可控
- **估值泡沫争议**：395亿美元估值远超收入，被质疑"炒概念"
  - 反驳：对标Tesla早期，技术领先溢价

### 1.2 Tesla Optimus（美国）—— 资源雄厚的"降维打击者"

| 维度 | 详情 |
|------|------|
| **成立时间** | 2021年（Tesla内部项目） |
| **总部** | 美国德州奥斯汀 |
| **负责人** | Elon Musk |
| **人员规模** | 约500-1000人（Tesla机器人团队） |
| **组织形式** | 上市公司内部项目 |

**发展历程**:
- 2021年：AI Day首次公布Optimus概念
- 2022年：原型机Bumblebee和Optimus Gen 1亮相
- 2023年：展示行走、抓取等基础能力
- 2024年：Optimus Gen 2发布，自由度19个，可深蹲、做瑜伽
- 2025年：计划部署约1000台机器人，目标售价**2-3万美元**

**产品与技术**:
- **Optimus Gen 2**：19个自由度，使用Tesla自研执行器
- **技术路线**：复用自动驾驶技术栈（视觉感知+端到端神经网络）
- **应用场景**：Tesla工厂内部测试，未来面向家庭服务

**资金情况**:
- 作为Tesla内部项目，无需独立融资
- Tesla市值约8000亿美元（2025年）
- 摩根士丹利测算：每替代美国1%劳动力，可为Tesla股价带来100美元NPV

#### 核心优势：全栈资源整合

**1. 技术复用优势**
- **自动驾驶→机器人**：
  - 视觉感知：Occupancy Network识别三维空间
  - 端到端神经网络：从FSD直接迁移
  - 数据引擎：百万辆Tesla采集真实道路数据
  - Dojo超算：训练AI模型的算力支持
- **成本控制**：
  - 执行器自研：Tesla自行设计电机、减速器
  - 规模化采购：与Tesla电动车共享供应链
  - 目标售价：**$2-3万美元**（行业最低目标）

**2. 场景闭环**
- **内部测试场**：Tesla工厂是全球最大测试场
  - 2025年计划部署1000台Optimus
  - 真实工业环境24/7测试
- **数据飞轮**：
  - 机器人在工厂工作→采集数据→训练模型→性能提升→更多部署

**3. 长期愿景：劳动力替代**
- **Elon Musk预言**：
  - "Optimus将是Tesla最重要的产品"
  - "未来每个人都会有一个人形机器人"
  - "劳动力将变得不再稀缺"
- **市场规模**：
  - 全球劳动力市场数十万亿美元
  - 美国制造业工人约1200万
  - 替代1%劳动力=100亿美元市场

#### 争议与挑战
| 问题 | 表现 | 影响 |
|------|------|------|
| **进展缓慢** | 2021年发布概念，2025年仍在测试 | 被质疑"PPT造车" |
| **技术路线** | 复用自动驾驶，但机器人≠汽车 | 环境更复杂，成功率待验证 |
| **竞争压力** | Figure AI、Agility已领先 | 时间窗口缩小 |

**结论**：Tesla Optimus是"慢热型"选手，短期被创业公司超越，但长期凭借资源优势和成本控制能力，仍是最大变量。

### 1.3 Boston Dynamics（美国）—— 技术标杆的转型之路

| 维度 | 详情 |
|------|------|
| **成立时间** | 1992年（MIT实验室孵化） |
| **总部** | 美国马萨诸塞州 |

| **人员规模** | 约500人 |
| **组织形式** | 上市公司子公司（现代汽车） |

**发展历程**:
- 1992年：从MIT Leg Lab孵化成立
- 2013年：被Google收购
- 2017年：被SoftBank收购
- 2020年：四足机器人Spot商业化
- 2021年：被现代汽车集团以11亿美元收购
- 2024年4月：发布全电动Atlas，取代液压版
- 2026年1月：现代汽车宣布计划2028年量产Atlas

**产品与技术**:
- **Atlas**：液压版（退役）→ 电动版（2024），28个自由度
- **Spot**：四足机器人，已商业化
- **技术路线**：传统控制+机器学习，强调运动控制能力
- **应用场景**：研究、工业巡检、物流

**资金情况**:
- 2021年被现代汽车收购，估值11亿美元
- 现代汽车计划投资4.8亿美元建设量产体系

#### 技术深度：行业标杆

**1. 液压→电动转型**
```
Atlas进化路线：
2013 液压Atlas → 2016 Atlas二代 → 2024 电动Atlas
   ↓              ↓                ↓
  原型机        跑酷/后空翻        商业化准备
```
- **液压版退役**：2024年4月，波士顿动力宣布液压Atlas退役
- **电动版发布**：新一代全电动Atlas，更安静、更清洁、更易维护

**2. 运动控制能力**
- **跑酷级动态平衡**：后空翻、跳跃、翻越障碍
- **全身协调**：28个自由度协同控制
- **世界模型**：预测自身运动和环境交互

**3. Spot商业化经验**
- **全球部署**：超千台Spot在电力、建筑、安保领域运行
- **收入模式**：硬件销售+软件订阅+服务合同
- **盈利情况**：波士顿动力称Spot业务已实现盈利

#### 商业化困境与转型
| 时期 | 困境 | 转型 |

| Google时期(2013-2017) | 技术炫但无产品 | 被转手给软银 |
| 软银时期(2017-2021) | 持续亏损 | 被转手给现代 |
| 现代时期(2021-至今) | 寻求商业化路径 | Spot盈利，Atlas量产准备 |

**现代汽车战略**：
- **2028年Atlas量产目标**
- **应用场景**：汽车制造（现代全球工厂）
- **投资计划**：4.8亿美元建设制造体系

#### 行业地位：技术天花板
- **学术界地位**：人形机器人运动控制的"教科书"
- **产业影响**：Figure AI、Agility等创始人多来自波士顿动力 alumni
- **技术输出**：控制算法、仿真平台（Isaac Sim等借鉴其经验）

### 1.4 Agility Robotics（美国）—— 首家商业化人形机器人公司

| 维度 | 详情 |
|------|------|
| **成立时间** | 2015年 |
| **总部** | 美国俄勒冈州奥尔巴尼 |
| **创始人** | Damion Shelton（CEO）、Jonathan Hurst（CTO，俄勒冈州立大学教授） |
| **人员规模** | 约200人 |
| **组织形式** | 创业公司（独角兽） |
| **核心标签** | **全球首家实现商业化的人形机器人公司** |

#### 发展历程

**技术积累期（2015-2018）**:
- 2015年：从俄勒冈州立大学动态机器人实验室孵化成立
- 2017年：发布Cassie双足平台（无上身，专注下肢行走算法）
- 2018年：完成种子轮800万美元融资

**产品化期（2019-2023）**:
- 2019年：推出首代Digit（基于Cassie增加上身和手臂）
- 2020年：完成A轮2025万美元融资，交付首批两台Digit给福特汽车
- 2022年：完成B轮1.5亿美元融资（Playground Global、亚马逊产业创新基金领投）
- 2023年：发布Digit V4，与GXO Logistics开始试运营

**商业化突破期（2024-2025）**:
- 2024年3月：完成C轮1.5亿美元融资，估值**10亿美元**，跻身独角兽
- 2024年6月：与GXO达成**多年战略合作协议**，首次实现人形机器人商业运营
- 2024年10月：在Spanx工厂完成"实习期"，正式投入生产作业
- 2024年11月：与**舍弗勒集团**签订战略投资+采购协议，计划在全球100家工厂部署
- 2025年：以**17.5亿美元估值**寻求4亿美元新一轮融资（WP Global、软银参与）
- 2025年3月：发布**新一代Digit**，负载提升至23kg，续航超6小时，外观更轻量化

#### 产品技术详解：Digit系列

**产品演进路线**:
```
Cassie (2017) → Digit V1 (2019) → V2/V3 (2021-2022) → V4 (2023) → 新一代 (2025)
     ↓              ↓                    ↓                ↓              ↓
  纯下肢平台    首次增加上身       增加末端执行器    优化人机交互    全面提升性能
```

**Digit V4/新一代技术规格**:
| 参数 | 规格 |
|------|------|
| **身高** | 5英尺9英寸（约175cm） |
| **体重** | 约65-70kg（新一代"瘦身"设计） |
| **负载能力** | V4: 16kg → 新一代: **23kg**（提升43%） |
| **续航时间** | 新一代: **6小时以上**（V3约4小时） |
| **行走速度** | 约1.5m/s（人类正常步速） |
| **自由度** | 总计约20+ DOF：手臂4×2 + 躯干 + 腿部 + 脚部2 |
| **特色设计** | **鸵鸟腿**（向后弯曲膝关节），提升行走稳定性 |
| **环境适应性** | 密封接头，可户外操作；通过UN 38.3航空货运认证 |
| **感知系统** | 头部集成摄像头、LiDAR；上身多传感器融合 |
| **交互能力** | V4起增加"眼睛"和表情显示，支持人机互动 |

**技术路线特点**:
1. **模块化设计**：关节、手臂、躯干均可快速更换维修
2. **专用AI而非通用大模型**：针对物流场景优化的任务专用模型
3. **保守但稳定**：动作相对缓慢（视频显示"小碎步"），但步态稳定可靠
4. **基于NVIDIA平台**：使用Isaac Sim和Isaac Lab进行全身控制基础模型训练

#### 商业模式创新：RaaS (Robot-as-a-Service)

Agility是**首家采用RaaS模式**的人形机器人公司，这使其成为"全球首家商业化人形机器人公司"。

**定价策略**:
| 模式 | 价格 | 说明 |
|------|------|------|
| **购买** | 约**25万美元/台** | 一次性买断 |
| **RaaS租赁** | 按需付费 | 主流模式，含硬件+软件+维护 |

**RaaS服务包（Agility Arc平台）**:
- ✅ Digit机器人硬件
- ✅ Agility Arc云软件平台（任务编排、远程监控）
- ✅ 全天候技术支持
- ✅ 预防性维护和维修
- ✅ 软件更新和功能升级

**ROI计算**:
- Agility宣称Digit可工作**20,000小时**（约5年每天8小时）
- 按美国仓储工人时薪$15-20计算，20,000小时人工成本约$30-40万
- Digit售价$25万，RaaS模式成本更低，具有明确经济价值

## 九、结论与建议

### 9.1 核心结论

1. **行业处于爆发前夜**：2025年是量产元年，2026-2027年将迎来商业化拐点
2. **技术路线尚未收敛**：端到端VLA vs 分层控制，各有优劣
3. **国内外差距缩小**：中国在硬件、成本、供应链方面具备优势
4. **估值存在泡沫**：多数公司估值远超收入，需警惕回调风险
5. **应用场景从工业开始**：家庭服务仍需3-5年技术积累

### 9.2 投资建议

**对投资者**:
- 关注有明确订单和量产能力的企业
- 警惕纯概念、无产品的公司
- 重点关注VLA模型、灵巧手、低成本执行器等核心技术

**对从业者**:
- 端到端AI+机器人是趋势，需强化AI能力
- 工业场景是当前最优落地路径
- 成本控制和供应链能力是关键竞争力

**对政策制定者**:
- 加快安全标准和法规建设
- 支持关键零部件国产化
- 引导产业有序发展，防止泡沫化
