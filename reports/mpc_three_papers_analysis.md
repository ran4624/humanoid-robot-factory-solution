## 一、论文概述与核心贡献

### 1.1 WMPC (World Model Predictive Control with Multimodal Adaptation)

**发表**: ICLR 2025  
**核心思想**: 将世界模型与模型预测控制(MPC)相结合，利用预训练的可微世界模型预测系统动态并优化控制动作

**主要贡献**:
- 提出统一框架，整合多模态状态表示、任务特定成本学习和约束感知优化
- 消除RL-based世界模型需要大量策略训练的需求
- 引入"弹性模型更新"机制，平衡短期修正与长期知识保留
- 实现记忆增强微调和通用技能提升

**技术特点**:
```
┌─────────────────────────────────────────────────────────────┐
│                    WMPC Framework                            │
├─────────────────────────────────────────────────────────────┤
│  多模态输入 → 预训练可微世界模型 → MPC优化 → 控制动作        │
│       ↓                                                        │
│  弹性模型更新                                                  │
│  • 短期修正: 即时反应调整                                      │
│  • 长期保留: 知识持久化                                        │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 DDCQ-MPC (Data-Driven Constrained Quadratic MPC)

**发表**: BADS 2025  
**核心思想**: 针对工业机械臂的鲁棒实时MPC，通过数据驱动学习动态模型

**主要贡献**:
- 数据驱动的约束二次MPC框架
- 适用于工业机械臂的高频实时控制
- 结合学习动态与传统MPC的稳定性保证
- 处理约束条件下的优化问题

**技术特点**:
- 使用递归最小二乘(RLS)算法在线辨识参数
- 自适应遗忘因子策略
- 高效实现的数据驱动MPC

### 1.3 Sensitivity-Aware MPC

**发表**: IEEE T-RO 2025  
**作者**: Tommaso Belvedere, Marco Cognetti, Giuseppe Oriolo, Paolo Robuffo Giordano  
**核心思想**: 针对参数不确定性的机器人鲁棒MPC

**主要贡献**:
- 计算高效的鲁棒MPC方案
- 处理非线性系统中的参数不确定性
- 灵敏度感知设计提升鲁棒性

**技术特点**:
- 考虑参数变化高达 ±30% 的鲁棒性边界
- 形式化收敛保证
- 严格的理论分析

## 四、DDCQ-MPC 深度分析

### 4.1 数据驱动动态学习

**核心算法 - 递归最小二乘(RLS)**:
```
参数更新: θ(t) = θ(t-1) + K(t)[y(t) - φ^T(t)θ(t-1)]
增益矩阵: K(t) = P(t-1)φ(t)[λ + φ^T(t)P(t-1)φ(t)]^{-1}
协方差更新: P(t) = [I - K(t)φ^T(t)]P(t-1)/λ

其中:
  θ: 动态参数
  λ: 遗忘因子(自适应)
  φ: 回归向量
  y: 观测输出
```

**自适应遗忘因子**:
- 参数变化大时: 减小λ，快速适应
- 参数稳定时: 增大λ，保持稳定

### 4.2 约束二次规划形式

**标准QP问题**:
```
minimize   (1/2)x^T Q x + c^T x
subject to A_eq x = b_eq
           A_ineq x ≤ b_ineq
           x_min ≤ x ≤ x_max

其中 x 包含: 控制输入序列
```

**优势**:
- 凸优化，全局最优保证
- 存在高效求解器(OSQP, qpOASES)
- 硬约束严格满足

### 4.3 工业级实时实现

**性能指标**:
- 控制频率: 100-1000 Hz
- 求解时间: < 1ms (小规模问题)
- 内存占用: 可预测，适合嵌入式

## 五、Sensitivity-Aware MPC 深度分析

### 5.1 参数不确定性建模

**不确定性来源**:
- 负载质量变化
- 摩擦系数变化
- 几何参数误差
- 温度影响

**数学描述**:
```
系统动态: ẋ = f(x, u, p)
其中 p ∈ [p_min, p_max] 为不确定参数

灵敏度方程: ∂x/∂p (传播参数变化影响)
```

### 5.2 灵敏度传播

**前向灵敏度传播**:
```
d/dt(∂x/∂p) = ∂f/∂x · ∂x/∂p + ∂f/∂p

计算复杂度: O(n_x · n_p)
其中 n_x: 状态维度, n_p: 参数维度
```

**在MPC中的应用**:
1. 预测轨迹的不确定性边界
2. 保守但安全的约束处理
3. 鲁棒成本函数设计

### 5.3 鲁棒性保证

**理论结果**:
- **输入-状态稳定性(ISS)**: 证明闭环系统稳定
- **约束满足**: 在参数变化±30%内保证约束
- **收敛性**: 递归可行性和稳定性证明
